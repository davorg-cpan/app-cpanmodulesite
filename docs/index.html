App-CPANModuleSite

Automatically create a web site for a CPAN module.

Current version: v0.0.8

Gabor asked if more CPAN modules should have their own web sites.

I replied, pointing out how easy it is to create project web sites on GitHub Pages.

But I wondered if we could make it even easier. This module is the result.
